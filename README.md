# LessElegantNote

## 项目简介
一个基于Typst的笔记模板。

## 学习资源
* [Typst官方文档](https://typst.app/docs)

## 使用指南
1. 下载本项目并解压。
2. 在vscode中打开项目文件夹。
3. 安装vscode插件 Typst LSP 和 Typst Preview。
4. 参照/template/notebook.typ，创建你自己的笔记，按 “ctrl + k,v” 即可预览。
